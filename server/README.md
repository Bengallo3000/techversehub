# TechVerseHub – React + Node Crypto Shop
Display EUR/USD, pay in crypto (BTC, ETH, ...).  
Admin panel, Telegram bot, photo upload, themes.

## Quick Start (local)
1. Clone repo
2. `npm install` (root)
3. Copy `server/.env.example` → `server/.env` and fill keys
4. `npm run dev` → http://localhost:3000
